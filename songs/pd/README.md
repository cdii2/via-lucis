# songs/pd — public-domain starter pack

Generated by `generate.py` (raw SMF, no deps — run `python generate.py` to
rebuild). Format 1, 480 tpq, track "Right" = melody, track "Left" = simple
bass, velocity 80.

- `ode-to-joy.mid` — Beethoven, Hymn to Joy theme; C major, 16 bars.
- `amazing-grace.mid` — "New Britain" hymn tune; G major 3/4, full 16-bar hymn.
- `minuet-in-g.mid` — Petzold (BWV Anh. 114); G major, complete 32 bars,
  simplified original bass (use the trainer's loop instead of repeats).
- `vivaldi-largo.mid` — Vivaldi, Winter (RV 297) 2nd mvt; complete 18-bar solo
  melody from the Mutopia urtext, transposed Eb→C, bass reduced to roots.
- `clair-de-lune.mid` — Debussy, Suite Bergamasque; the famous opening
  (urtext bars 1–9), transposed Db→C, inner voices thinned. The full piece in
  Db stays a Tier 3 aspiration.

These are public domain works; these original arrangements are dedicated to
the public domain under CC0.
