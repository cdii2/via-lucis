WiFi

WiFi changes apply after reboot. The password is write-only — the device never sends it back, so this field always shows blank; the placeholder tells you whether one is stored. Leave it blank to keep the current password. Type a new one and it's saved like any other setting.

Backup

Save a copy of all settings (colors, ramp, calibration, WiFi) as a file you can re-import later by pasting into a settings PUT, or just keep as a record of your calibration numbers.
